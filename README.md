# Chinook Music Store — SQL Sales Analysis
Author: **Vu, Thi Thao Nguyen**

This project analyzes the Chinook digital music store database using SQL to uncover trends in genres, artists, albums, customers, and markets.  
It also demonstrates SQL skills including JOINs, aggregations, CTEs, and window functions.

---

## Project Structure
chinook_analysis.sql → all SQL queries used in the analysis.  
docs/INSIGHTS.md → detailed insight report and business recommendations.


---

## Analytical Focus
- Top-performing & low-performing genres  
- Genre efficiency (revenue per track)  
- Best-selling albums  
- Highest-earning artists  
- Biggest markets by country  
- Most valuable customers  
- Yearly revenue trends

## Full Insight Report  
See **docs/INSIGHTS.md**

## SQL Techniques Used
- Multi-table JOINs  
- Aggregate functions (SUM, COUNT, ROUND)  
- GROUP BY + ORDER BY  
- DISTINCT  
- Common Table Expressions (CTE)  
- Window Functions: `LAG()` for year-over-year growth  

---

## Dataset
- Chinook sample database (converted to MySQL)  
- Includes: Invoice, InvoiceLine, Track, Album, Artist, Genre, Customer

Showcase SQL analytical capability, data storytelling, and business insight generation.
