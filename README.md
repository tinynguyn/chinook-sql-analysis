# Chinook Music Store — SQL Sales Analysis
Author: **Vu, Thi Thao Nguyen**

This project analyzes the Chinook digital music store database using SQL to uncover trends in genres, artists, albums, customers, and markets.  
It also demonstrates SQL skills including JOINs, aggregations, CTEs, and window functions.

---

## Project Structure
chinook_analysis.sql → all SQL queries used in the analysis.  
docs/INSIGHTS.md → detailed insight report and business recommendations.
